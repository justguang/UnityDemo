# UnityDemo


        游戏说明：PC端经典的 FPS打僵尸游戏，场景中多处固定点随机生成僵尸；
		 靠近或攻击僵尸会吸引其仇恨，导致其朝玩家追逐一段时间，仇恨消失则停止追逐；
		 玩家被僵尸攻击到扣除相应血量，如果玩家自身血量为0=>死亡，游戏结束。	

    操作：
	    移动：WSAD
	    换弹：R
	    快速移动：Shift + WSAD
	    起跳：空格键
	    射击：鼠标左键
	    瞄准：鼠标右键
	    功能键：Esc
	    【P键 扣除自身血量】
